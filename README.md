# ☕ Final Project: Codédex Java

Este é o projeto final da trilha de Java da plataforma **Codédex**. O objetivo deste desafio foi aplicar os fundamentos da linguagem, com foco em lógica estruturada e nos pilares da Programação Orientada a Objetos (POO).

---

## 🇧🇷 Português

### Descrição
O projeto representa a conclusão da jornada de aprendizado em Java, uma das linguagens fundamentais para o desenvolvimento de sistemas robustos e escaláveis. Durante o desafio, foram explorados conceitos que vão desde a sintaxe básica até a organização de código utilizando classes e métodos, preparando a base para o desenvolvimento de APIs e aplicações de grande porte.

### Tecnologias Utilizadas
* **Java (JDK 17 ou superior)**: Linguagem principal para o desenvolvimento da lógica.

### O que este projeto demonstra:
* **Programação Orientada a Objetos (POO)**: Aplicação de conceitos como classes, objetos e métodos.
* **Tipagem Forte e Estática**: Gerenciamento eficiente de tipos de dados e memória.
* **Lógica de Programação**: Implementação de algoritmos para resolução de problemas propostos pela trilha.
* **Base para Ecossistema Enterprise**: Domínio da sintaxe necessária para evoluir para frameworks como Spring Boot.

## 🇺🇸 English

### Description
This is the final project for the Java track on the **Codédex** platform. The goal of this challenge was to apply language fundamentals, focusing on structured logic and the pillars of Object-Oriented Programming (OOP).

### Technologies Used
* **Java (JDK 17 or higher)**: Core language used for logic development.

### Key Takeaways:
* **Object-Oriented Programming (OOP)**: Application of concepts such as classes, objects, and methods.
* **Strong and Static Typing**: Efficient management of data types and memory.
* **Programming Logic**: Implementation of algorithms to solve problems proposed by the track.
* **Enterprise Ecosystem Foundation**: Mastery of the syntax required to advance toward frameworks like Spring Boot.
